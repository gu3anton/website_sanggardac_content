# Sanggar DAC — Public Content

Repository ini adalah penyimpanan publik untuk konten runtime website Sanggar DAC.

## Struktur

- `content/site.json` — profil, kelas, jadwal, galeri, dan informasi pendaftaran.
- `public/gallery/` — gambar galeri yang ditampilkan pada website.

## Keamanan dan privasi

- Jangan menyimpan token, secret, data anggota, informasi kesehatan, atau data pribadi di repository ini.
- Unggah hanya foto yang telah mendapat izin publikasi.
- Seluruh file dan riwayat commit di repository ini dapat dilihat publik.
- Source code website disimpan terpisah pada repository private.

Konten dikelola melalui CMS Sanggar DAC.
